# 27 移除元素 easy

## 题目

给你一个数组 `nums` 和一个值 `val`，你需要 **[原地](https://baike.baidu.com/item/原地算法)** 移除所有数值等于 `val` 的元素。元素的顺序可能发生改变。然后返回 `nums` 中与 `val` 不同的元素的数量。

假设 `nums` 中不等于 `val` 的元素数量为 `k`，要通过此题，您需要执行以下操作：

- 更改 `nums` 数组，使 `nums` 的前 `k` 个元素包含不等于 `val` 的元素。`nums` 的其余元素和 `nums` 的大小并不重要。
- 返回 `k`。

**示例 1：**

```js
输入：nums = [3,2,2,3], val = 3
输出：2, nums = [2,2,_,_]
解释：你的函数应该返回 k = 2, 并且 nums 中的前两个元素均为 2。
你在返回的 k 个元素之外留下了什么并不重要（因此它们并不计入评测）。
```

**示例 2：**

```js
输入：nums = [0,1,2,2,3,0,4,2], val = 2
输出：5, nums = [0,1,4,0,3,_,_,_]
解释：你的函数应该返回 k = 5，并且 nums 中的前五个元素为 0,0,1,3,4。
注意这五个元素可以任意顺序返回。
你在返回的 k 个元素之外留下了什么并不重要（因此它们并不计入评测）。
```

**提示：**

- `0 <= nums.length <= 100`
- `0 <= nums[i] <= 50`
- `0 <= val <= 100`

-------

## 解题思路

> 不愧是简单题，不过第一时间还是想用工具函数去处理，直接map...或者filter...
>
> 但算法的解决办法绝对不是这样的。
>
> 而本题考察的就是快慢指针。

![](F:\JavaScript-Charles\LeetCode\images\27移除元素-1.png)

```js
/**
 * @param {number[]} nums
 * @param {number} val
 * @return {number}
 */
var removeElement = function(nums, val) {
    let i = 0;
    for (let j = 0; j < nums.length; j++) {
        if (nums[j] !== val) {
            nums[i] = nums[j];
            i++;
        }
    }
    return i;
};
```

> 题目要求返回`k` ，那我们只需要知道有几个是满足不等于`val` 的条件的即可。
>
> 定义一个慢指针`i`， 通过快指针`nums` 的循环下标，挨个判断是否和`val`相等
>
> 题目只要求记录不同的，所以循环内判断`nums[j]!==val`， 满足条件则让`nums[i]= nums[j]`， 即可替换位置。满足条件后让慢指针递增。
>
> 因为题目要求正确更改 `nums` 数组，使 `nums` 的前 `k` 个元素包含不等于 `val` 的元素。`nums` 的其余元素和 `nums` 的大小并不重要。
>
> 所以我们只需要关注前面我们筛选出来的值即可
>
> 比如`nums=[3,2,2,3]，val=3`，则最后输出`i=2， nums=[2,2,23]` ，i为2，所以我们只关注`nums`前两位
